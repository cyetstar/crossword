# 填字游戏解答工具

一个基于Vue3的填字游戏解答工具，可以帮助你根据文章内容找到正确的单词。

## 功能特点

1. **文章单词提取**：从粘贴的纯文本中提取所有单词
2. **智能筛选**：根据字母数量筛选单词
3. **交互式反馈**：点击字母设置状态（灰色/黄色/绿色）
4. **智能推荐**：根据反馈自动筛选候选单词
5. **历史记录**：保存已尝试的单词，支持重新开始

## 使用方法

### 第一步：提取单词
1. 在输入框中粘贴文章纯文本内容
2. 点击"提取单词"按钮
3. 系统会自动提取所有单词（至少3个字母）

### 第二步：开始游戏
1. 输入单词长度（如5）
2. 点击"开始游戏"
3. 系统会从单词库中随机选择一个单词显示

### 第三步：设置字母状态
点击每个字母框，循环切换状态：
- **灰色（⚫）**：该字母在答案中不存在
- **黄色（🟡）**：该字母存在，但位置不对
- **绿色（🟢）**：该字母正确且位置正确

### 第四步：获取下一个单词
1. 设置完所有字母状态后，点击"获取下一个单词"
2. 系统会根据你的反馈自动筛选候选单词
3. 重复步骤3-4，直到所有字母都是绿色

### 其他功能
- **选择候选单词**：点击候选单词列表中的任意单词，可以重新开始使用该单词
- **查看历史**：已尝试的单词会显示在历史记录中
- **重新开始**：点击历史记录中的单词，可以回到该单词并清除之后的反馈
- **清除所有**：点击"清除所有"按钮，清空当前游戏状态

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到Vercel

1. 将代码推送到GitHub
2. 在Vercel中导入项目
3. 构建命令：`npm run build`
4. 输出目录：`dist`
5. 自动部署完成

## 技术栈

- Vue 3
- TypeScript
- Vite
- 纯前端实现，无需后端
